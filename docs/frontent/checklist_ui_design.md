### **Система автосохранения**
```
┌─ Логика автосохранения ────────────────────────────────────┐
│                                                            │
│ 🔄 Триггеры сохранения:                                    │

│ • При смене вопроса                                        │

│ • Перед закрытием браузера                                 │
│                                                            │
│ 💾 Версионность:                                           │
│ • Основное сохранение: каждые 15 сек                       │

│ • Снапшоты сессий: при запуске                             │
│ • История изменений: последние 50 действий                 │
│                                                            │
│ 📶 Оффлайн-режим:                                          │
│ • LocalStorage: текущее состояние                          │
│ • IndexedDB: история и снапшоты                            │
│ • Синхронизация при восстановлении связи                   │
└────────────────────────────────────────────────────────────┘
```


### **Система уведомлений**
```
┌─ Типы уведомлений ─────────────────────────────────────────┐
│                                                            │
│ 💾 Статусные (правый верх, 3 сек):                         │
│ • "Сохранено"           - зеленый                          │
│ • "Ошибка сохранения"   - красный                          │
│ • "Восстановлено из автосохранения" - синий                │
│                                                            │
│ 🎯 Достижения (центр, 5 сек + звук):                       │
│ • Завершение раздела                                       │
│ • Достижение контрольных точек                             │ 
│ • Полное завершение чеклиста                               │
└────────────────────────────────────────────────────────────┘
```

Эта детальная система обеспечивает **максимальную эргономику** при работе с длинными опросниками, делая процесс **осмысленным** и **не утомительным**.
